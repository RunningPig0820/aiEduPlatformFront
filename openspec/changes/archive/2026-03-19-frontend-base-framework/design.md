# 前端基础框架 - 设计文档

## 1. 角色与布局策略

### 1.1 角色层级

```
平台管理员（独立角色）
     │
     │ 管理多个组织
     ▼
┌─────────────┐
│  组织/学校  │
└─────────────┘
     │
     │ 学校管理员 = 老师角色 + 额外权限
     ▼
老师 / 家长 / 学生
```

### 1.2 布局差异

| 角色 | 布局类型 | 特点 |
|------|----------|------|
| 管理员 | AdminLayout（左右菜单） | 桌面优先，功能入口多 |
| 学生/老师/家长 | DashboardLayout（响应式卡片） | 移动优先，适配平板 |

### 1.3 布局组件架构

```
AdminLayout                    DashboardLayout
(管理员专用)                    (学生/老师/家长)
┌────┬──────────┐             ┌────┬──────────────┐
│菜单│ 内容+AI  │             │菜单│ 内容 + AI    │
└────┴──────────┘             └────┴──────────────┘

共享组件:
├─ AIChatPanel (右侧 AI 面板)
├─ Navbar (顶部导航栏)
└─ UserMenu (用户下拉菜单)
```

## 2. AI 助手面板设计

### 2.1 面板状态

**展开状态（约 320px）：**
```
┌────────────────────────┬────────────┐
│                        │            │
│                        │  AI 助手   │
│       页面内容         │            │
│                        │  [对话区]  │
│                        │            │
│                        │  [输入框]  │
└────────────────────────┴────────────┘
```

**收起状态（48px 窄条）：**
```
┌────────────────────────────┬──────────┐
│                            │          │
│                            │    AI    │
│       页面内容             │   对话   │
│                            │          │
│                            │    ▶     │ ← 展开按钮
└────────────────────────────┴──────────┘
```

### 2.2 设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 收起方案 | 48px 窄条 | 与 IDE 侧边栏效果一致，用户熟悉 |
| 可见性 | 始终可见 | 用户不会忘记 AI 助手存在 |
| 响应式 | 手机端可隐藏 | 最大化手机端内容区域 |

### 2.3 响应式适配

| 设备 | 面板行为 |
|------|----------|
| 桌面 (>1024px) | 面板可展开/收起，收起为窄条 |
| 平板 (768-1024px) | 面板可展开/收起，默认收起 |
| 手机 (<768px) | 面板隐藏，通过浮动按钮触发 |

## 3. 页面元信息系统

```javascript
const pageMeta = {
  code: 'STUDENT_HOME',           // 页面唯一标识
  name: '学生首页',
  description: '学生登录后的首页，展示学习概览、待办作业、错题提醒',
  features: [
    '查看学习统计数据',
    '查看待完成作业列表',
```

## 6. 技术决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| AI 对话流式输出 | SSE (EventSource) | 实时性好，实现简单 |
| 面板动画 | CSS transition | 性能好，兼容性强 |
| 元信息存储 | 前端常量文件 | 简单直接，易于维护 |
| 待开发提示 | Toast (daisyUI alert) | 与现有 UI 框架一致 |
