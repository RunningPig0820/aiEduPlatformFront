## ADDED Requirements

### Requirement: 用户应能通过关系图谱查看教材知识点与知识点的关联

系统应在中间区域使用 React Flow 渲染关系图谱，展示教材知识点与关联知识点之间的连接关系。

#### Scenario: 用户选择知识点后加载图谱
- **WHEN** 用户在左侧树中点击知识点节点
- **THEN** 中间区域显示加载中状态，请求该知识点的关联图谱数据并渲染

#### Scenario: 图谱展示两个核心节点及其关联
- **WHEN** 图谱数据加载完成
- **THEN** 图中展示「教材知识点」和「知识点」两个节点，以及它们之间的有向连线

#### Scenario: 图谱支持缩放和拖拽
- **WHEN** 用户在图谱区域内操作
- **THEN** 支持鼠标滚轮缩放、拖拽平移、节点拖拽

### Requirement: 图谱节点应支持点击选中

用户可以点击图谱中的节点，选中后右侧详情面板展示该节点信息。

#### Scenario: 用户点击图谱中的教材知识点节点
- **WHEN** 用户点击图谱中的教材知识点类型节点
- **THEN** 该节点高亮显示，右侧详情面板展示教材知识点详情

#### Scenario: 用户点击图谱中的知识点节点
- **WHEN** 用户点击图谱中的知识点类型节点
- **THEN** 该节点高亮显示，右侧详情面板展示知识点详情

### Requirement: 图谱区域应显示加载状态

在图谱数据加载过程中，应展示加载动画或占位符。

#### Scenario: 图谱数据加载中
- **WHEN** 用户选中知识点节点后图谱数据正在请求
- **THEN** 中间区域显示加载动画，数据返回后替换为图谱

### Requirement: 图谱节点应提供 Tooltip 悬停提示

鼠标悬停在图谱节点上时，应显示节点详细信息。

#### Scenario: 鼠标悬停显示节点信息
- **WHEN** 用户鼠标悬停在图谱节点上
- **THEN** 显示 Tooltip，包含节点完整名称和类型（教材知识点/知识点）
